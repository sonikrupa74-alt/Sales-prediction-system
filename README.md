# Sales Prediction System 🚀

## Overview

This project is a machine learning-based system that predicts future sales using historical data. It provides real-time predictions through a deployed FastAPI backend.

## Tech Stack

* Python
* pandas, NumPy, scikit-learn
* FastAPI
* Render (Deployment)

## Features

* Real-time sales prediction using API
* Data preprocessing and model training
* Deployed and accessible online

## Live Demo

https://sales-backend-dil3.onrender.com

## How It Works

1. User sends input data
2. FastAPI processes request
3. Model predicts sales
4. Result returned instantly

## Result

* Built and deployed a real-time ML system
* Demonstrates end-to-end workflow (data → model → API → deployment)
