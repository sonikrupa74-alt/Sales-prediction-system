# Sales Prediction System 🚀

## Overview

This project is a machine learning-based sales prediction system that forecasts future sales using historical business data. It provides real-time predictions through a FastAPI backend with frontend integration for user interaction.

## Tech Stack

* Python
* pandas, NumPy, scikit-learn
* FastAPI
* HTML, CSS, JavaScript
* Render (Deployment)

## Features

* Real-time sales prediction
* Data preprocessing and regression model training
* Frontend and backend integration
* REST API-based prediction system
* Deployed and accessible online

## How It Works

1. User enters sales-related input data
2. Frontend sends request to FastAPI backend
3. Machine learning model processes the data
4. Predicted sales output is returned instantly

## Result

* Built and deployed an end-to-end sales forecasting system
* Demonstrates complete ML workflow (data → preprocessing → model → API → frontend → deployment)
* Integrated frontend with backend for real-time prediction
